# Inventario de Farmacia – Juan

Sistema sencillo de inventario para una farmacia, desarrollado en JavaScript como práctica previa a Programación Orientada a Objetos (POO).

## Funcionalidades
- Registrar medicamentos
- Consultar inventario
- Vender medicamentos
- Calcular total de ventas del día

## Tecnologías usadas
- JavaScript
- Node.js

## Cómo ejecutar el programa
1. Abrir la terminal
2. Ir a la carpeta del proyecto
3. Ejecutar el comando:

```bash
node farmacia.js
